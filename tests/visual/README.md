# Visual Checkpoints

## Purpose

This folder documents the small screenshot checkpoint set used by the browser harness.

The generated images live in `output/playwright/checkpoints/` and are not committed.

## Current Checkpoints

- `desktop-full-motion.png`
- `desktop-reduced-motion.png`
- `mobile-full-motion.png`

## Review Intent

These checkpoints are a compact review aid for:

- intro completion state
- reduced-motion completion state
- one mobile viewport completion state

They supplement the smoke checks. They are not meant to replace broader visual QA as more scenes are implemented.
